Customer Segmentation Analysis using SQL

# Project Overview
This project focuses on analyzing customer demographic and behavioral data to
identify meaningful customer segments. The analysis helps businesses understand
customer profiles and target them effectively.

The project is built completely using **MySQL** and focuses on real-world SQL
analysis techniques used by data analysts.

---

# Tools & Technologies
- MySQL
- SQL (Joins, Aggregations, Subqueries, Window Functions)
- MySQL Workbench
- GitHub

# Dataset Information

- Work_Experience
- Spending_Score

# Analysis Breakdown

# Basic SQL Analysis
- Total number of customers
- Gender-wise customer distribution
- Average age of customers
- Segment-wise customer count
- Average family size

# Intermediate SQL Analysis
- Average age per customer segment
- Spending score distribution
- Profession-wise segmentation
- Married vs non-married segmentation
- Work experience analysis

# Advanced SQL Analysis
- Ranking customer segments by age
- Percentage contribution of each segment
- Identifying high-value customers
- Age group based segmentation
- Use of window functions

---

# How to Run This Project
1. Create database using `schema.sql`
2. Import `train.csv` into MySQL
3. Run queries from `sql_analysis` folder
4. Analyze results in MySQL Workbench
